## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
// 예시: 랜덤한 5글자 단어 선정
char names[5][20];
int scores[5];
int count = 0;  // 현재 학생 수
```

**방법 2: 구조체 사용 (선택)**
```c
// 예시: 학생 정보를 구조체로 묶어서 관리
struct Student {
    char name[20];
    int kor;
    int eng;
    int math;
    float avg;
};
struct Student students[5];  // 학생 5명 저장
```
